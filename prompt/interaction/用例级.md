请基于六边形架构，为 Quadra Interaction 服务实现一个完整的用例：

【用例名称】
LikeTarget（点赞目标）

【业务要求】
- 对目标（动态/视频）进行点赞
- 点赞必须幂等（同一用户同一目标只能点赞一次）
- 创建 Interaction 聚合根（action_type=LIKE）
- 发布 InteractionCreatedEvent（写入 outbox_event，供 content 域消费更新计数）

【技术要求】
- JDK 21
- Spring Boot 3
- MyBatis-Plus
- MySQL 8

【必须输出以下内容】
1. quadra-interaction-domain 模块：
   - Interaction 聚合根（包含 like / comment 工厂方法，内部必须校验非法状态）
   - TargetType / ActionType 值对象（枚举）
   - InteractionCreatedEvent / InteractionCanceledEvent

2. quadra-interaction-application 模块：
   - LikeTargetCommand
   - LikeTargetUseCase 接口
   - InteractionApplicationService 实现
   - port.out：InteractionRepositoryPort, EventPublisherPort

3. quadra-interaction-adapter-out 模块：
   - InteractionRepositoryImpl（单表持久化，处理 DuplicateKeyException 实现幂等）
   - OutboxEventRepositoryImpl（写事件表，与业务同事务）

4. quadra-interaction-adapter-in-web 模块：
   - InteractionController（POST /api/v1/interactions/likes 等）

【强约束】
- 不允许跨层调用
- 不允许出现 DAO 直连 Controller
- Repository 必须操作聚合根
- Domain 层不能出现 Spring 注解
- LIKE/COMMENT 必须统一在 interaction 表，禁止拆表

请严格按照分层结构输出代码。
