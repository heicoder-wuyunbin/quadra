请基于六边形架构，为 Quadra Social 服务实现一个完整的用例：

【用例名称】
FollowUser（用户关注）

【业务要求】
- 用户关注另一个用户（单向关注）
- 校验不能自己关注自己
- 创建 UserFollow 聚合根
- 支持双向好友场景：Friendship 同事务内写入 A→B 与 B→A 两条记录

【技术要求】
- JDK 21
- Spring Boot 3
- MyBatis-Plus
- MySQL 8

【必须输出以下内容】
1. quadra-social-domain 模块：
   - UserFollow 聚合根（包含 follow 工厂方法，内部必须校验不能自己关注自己）
   - Friendship 聚合根（双向好友）
   - UserMatchLike 聚合根（LIKE/DISLIKE 匹配）
   - UserVisitor 聚合根（访客记录）

2. quadra-social-application 模块：
   - FollowUserCommand
   - FollowUserUseCase 接口
   - SocialApplicationService 实现
   - port.out：FollowRepositoryPort, FriendshipRepositoryPort, EventPublisherPort

3. quadra-social-adapter-out 模块：
   - FollowRepositoryImpl（单表持久化 user_follow）
   - FriendshipRepositoryImpl（同事务内写入 A→B 与 B→A 两条记录）
   - OutboxEventRepositoryImpl（写事件表，与业务同事务）

4. quadra-social-adapter-in-web 模块：
   - SocialController（POST /api/v1/social/follows 等）

【强约束】
- 不允许跨层调用
- 不允许出现 DAO 直连 Controller
- Repository 必须操作聚合根
- Domain 层不能出现 Spring 注解

请严格按照分层结构输出代码。
