请基于六边形架构，为 Quadra User 服务实现一个完整的用例：

【用例名称】
RegisterUser（用户注册）

【业务要求】
- 使用手机号注册
- 密码使用 BCrypt 加密（在 application 层处理）
- 创建 User + UserProfile + UserSetting
- 发布 UserRegisteredEvent（写入 outbox_event）

【技术要求】
- JDK 21
- Spring Boot 3
- MyBatis-Plus
- MySQL 8

【必须输出以下内容】
1. quadra-user-domain 模块：
   - User 聚合根（包含 register 工厂方法，内部必须校验非法状态）
   - UserProfile / UserSetting 实体
   - UserRegisteredEvent

2. quadra-user-application 模块：
   - RegisterUserCommand
   - RegisterUserUseCase 接口
   - UserApplicationService 实现（包含 BCrypt 加密处理）
   - port.out：UserRepositoryPort, EventPublisherPort

3. quadra-user-adapter-out 模块：
   - UserRepositoryImpl（拆三表持久化，处理 DuplicateKeyException 兜底）
   - OutboxEventRepositoryImpl（写事件表，与业务同事务）

4. quadra-user-adapter-in-web 模块：
   - UserController

【强约束】
- 不允许跨层调用
- 不允许出现 DAO 直连 Controller
- Repository 必须操作聚合根
- Domain 层不能出现 Spring 注解

请严格按照分层结构输出代码。
